# 📇 Blog Post Card - Componente UI

Este proyecto consiste en una **Tarjeta de Publicación de Blog** responsiva y estilizada, creada para practicar el modelo de caja (Box Model), diseño de componentes y efectos de interacción en CSS.

## 🚀 Vista Previa / DEMO
> [!TIP]
> https://jcarocav.github.io/fcc-cards/


## 🛠️ Tecnologías y Conceptos Aplicados
En este desafío de **freeCodeCamp**, apliqué los siguientes conceptos técnicos:

*   **HTML5 Semántico**: Uso de contenedores `div`, encabezados `h2` y enlaces `a`.
*   **CSS Box Model**: Manejo preciso de `margin`, `padding` y `border-radius`.
*   **Diseño de Componentes**: Creación de una tarjeta con desbordamiento controlado (`overflow: hidden`) para que la imagen se adapte a las esquinas redondeadas.
*   **Pseudo-clases**: Implementación de estados `:hover` en botones para mejorar la experiencia de usuario (UX).

## 🎨 Características del Diseño
*   **Fondo adaptable**: La tarjeta destaca sobre un fondo de página con contraste.
*   **Estilo de Botón**: Transformación de un enlace estándar en un botón interactivo mediante `display: inline-block`.
